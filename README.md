# Web Code Editor 🚀

## Overview
A browser-based code editor inspired by VS Code that allows users to write and preview HTML, CSS, and JavaScript in real time.

## Features
- Live preview of code  
- Separate editors for HTML, CSS, and JavaScript  
- Instant rendering of user input  
- Clean and responsive UI  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## How to Use
1. Open `index.html`
2. Write code in the editors
3. See output instantly in the preview panel
